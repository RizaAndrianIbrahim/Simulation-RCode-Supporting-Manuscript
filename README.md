This repository contains the simulation code used in the manuscript.

1. simulation_code.R

How to reproduce results:
1. Install R (version 4.0 or higher).
2. Run simulation_code.R.
3. Output data will be generated automatically.

No human participants or confidential data are involved.
All data are generated via stochastic simulation.
